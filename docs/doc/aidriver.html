driveVehicleBySteeringAngle(dt, moveForwards, steeringAngleNormalized, turnLeft, maxSpeed)¶

Drive vehicle by using a steering angle. This is similar to the Giants AIVehicleUtil.driveInDirection() but

instead of the direction (lx, lz) uses a steering angle.

Parameters

  • dt (number) – dt

  • moveForwards (boolean) – if true, we want the vehicle to move forwards, false for backwards

  • steeringAngleNormalized (number) – between 0 and 1, 1 being the maximum steering angle.

  • turnLeft (boolean) – true when turning to the left

  • maxSpeed (number) – speed we want the vehicle to drive

startCourseWithPathfinding(course, ix, zOffset, fieldNum, alwaysUsePathfinding)¶

PATHFINDING

Start course with pathfinding Will find a path on a field avoiding fruit as much as possible from the current position to waypoint ix of course and start driving. When waypoint ix of course reached, switch to the course and continue driving.

If no path found will use an alignment course to reach waypoint ix of course.

Parameters

  • course (Course) –

  • ix (number) –

  • zOffset (number) – or nil length offset of the goal from the goalWaypoint

  • fieldNum (number) – or nil if > 0, the pathfinding is restricted to the given field and its vicinity. Otherwise the

  • alwaysUsePathfinding (boolean) – use pathfinding even when close to target

Returns

true when a pathfinding successfully started or an alignment course was added

Return type

boolean

driveToPointWithPathfinding(waypoint, zOffset, allowReverse, course, ix, fieldNum)¶

Start driving to a point with pathfinding

Will find a path on a field avoiding fruit as much as possible from the current position to the given coordinates. Will call onEndCourse when it reaches that point, so you’ll have to have your own implementation of that

If no path is found, onNoPathFound() is called, you’ll need your own implementation of that to handle that case.

pathfinding considers any collision-free path valid, also outside of the field.

Parameters

  • waypoint (Waypoint) – The destination waypoint (x, z, angle)

  • zOffset (number) – length offset of the goal from the goalWaypoint

  • allowReverse (boolean) – allow reverse driving

  • course (Course) – course to start after pathfinding is done, can be nil

  • ix (number) – course to start at after pathfinding, can be nil

  • fieldNum (number) – if > 0, the pathfinding is restricted to the given field and its vicinity. Otherwise the

Returns

true when a pathfinding successfully started

Return type

boolean
